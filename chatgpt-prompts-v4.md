# Japan Night - 画像生成プロンプトガイド v4

## 共通スタイルルール
すべての画像で以下を守ること：
- **2頭身チビキャラ、太い黒の輪郭線、フラットでかわいいイラスト**
- **背景は完全に透明（PNG）**
- **サイズ: 1024x1024**
- 影なし、グラデーション最小限
- 色はパステル〜ビビッドでかわいい印象

---

## STEP 1: 顔パーツ

以下のプロンプトで頭部のみ生成：

```
チビキャラの顔・頭だけを描いてください（体・首・肩は描かない）。
スタイル：2頭身チビキャラ、太い黒の輪郭線、フラットでかわいいイラスト。
頭は髪と顔のみで、あごの下で切れるようにしてください。
背景：完全に透明（PNG）。
サイズ：1024x1024、中央配置。
[参考写真を添付して] この人の顔に似せて描いてください。
```

## STEP 2: 衣装ボディ

⚠️ 肌色は3パターン生成すること！衣装1つにつき3回生成します。

以下のプロンプトでボディのみ生成：

```
チビキャラの体だけを描いてください。[衣装名]を着ています（頭・顔は描かない。首の部分は空白／透明にする）。
スタイル：2頭身チビキャラ、太い黒の輪郭線、フラットでかわいいイラスト。
首から足先まで、腕と手も含めて描いてください。
キャラクターは正面を向いて立ち、腕は軽く横に広げた状態。
肌の色：[肌色指定]
背景：完全に透明（PNG）。
サイズ：1024x1024、中央配置。
```

### 肌色パターン（[肌色指定]に入れる）：

| ID | パターン | プロンプトに書く内容 |
|---|---|---|
| light | 色白 | 色白で明るい肌色（東アジア系の明るい肌） |
| medium | 中間 | 自然な小麦色の肌（中間的な肌色） |
| dark | 濃い | 濃い褐色の肌（ダークスキン） |

例：着物 × 色白 → `outfit_kimono_light.png`
例：着物 × 中間 → `outfit_kimono_medium.png`
例：浴衣 × 濃い → `outfit_yukata_dark.png`

アイテムだけを描く（手やキャラクターは描かない）。
アイテムは少し斜めに傾けて、自然な角度で。
背景：完全に透明（PNG）。
サイズ：1024x1024。アイテムをキャンバスの中央に配置。

## STEP 5: 背景

背景はキャラクターの後ろに表示します。日本をテーマにしたシーンを生成してください。

```
日本をテーマにした背景画像を描いてください。[シーン名]のシーンです。
スタイル：パステルカラーで柔らかく、かわいいイラスト風。太い輪郭線は不要。
キャラクターは描かないでください。背景のみ。
中央付近にキャラクターが立つスペースを空けてください（物を置かない）。
地面は下から15〜20%の位置に。
背景：透明ではなく、しっかり色を塗ってください。
サイズ：1024x1024。
```

## 生成の順番（おすすめ）

1. まず背景を2-3個試す（sakura, shrine, festival）
2. アイテムを2-3個試す（fan, onigiri, dango）
3. バディを1-2個試す（pikachu, totoro）
4. 位置・サイズを確認してOKなら残りを生成
5. 最後に衣装バリエーションを生成

---

## Python自動配置パラメータ

生成した画像はPythonスクリプトで自動的に正しい位置・サイズに調整されます。

### アイテムの配置：
- サイズ：キャンバスの15%
- 位置：左手付近（x=0.32, y=0.68）

### バディの配置：
- サイズ：キャンバスの30〜35%
- 位置：右横（x=0.82, y=0.88基準で足元合わせ）

### 背景の配置：
- サイズ：1024x1024（そのまま使用）
- 一番下のレイヤーとして描画
